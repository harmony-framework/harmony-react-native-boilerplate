## Prerequisites
-  View [MkDocs documentation](http://www.mkdocs.org/).

## Installation
In order to manually install MkDocs you'll need Python installed on your system, as well as the Python package manager, pip.

- pip install mkdocs
- pip install mkdocs-material
- cd docs
- mkdocs serve
